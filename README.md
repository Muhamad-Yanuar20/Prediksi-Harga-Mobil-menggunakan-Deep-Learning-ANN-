# Prediksi-Harga-Mobil-menggunakan-Deep-Learning-ANN-
Notebook ini berisi model deep learning berbasis Artificial Neural Network (ANN) untuk memprediksi harga mobil menggunakan Automobile Dataset.  Arsitektur yang digunakan:  **Deeper ANN with Dropout (3 Hidden Layers + Regularization) Metrik Evaluasi: RMSE · MAE · MAPE · R²  Skor : RMSE: 6,366  RMSE: 6,366 MAE: 3,968 MAPE: 19.96% R²: 0.6687
